# Codex Skills Collection

This repository contains a curated set of custom Codex skills for UI design, frontend structure, motion, color, and pattern guidance.

## Included Skills

- `frontend-skill`: Build visually strong landing pages, websites, apps, prototypes, demos, and game UI.
- `premium-ui-polish`: Refine interface quality with layered shadows, premium easing, typography tuning, and restrained glass effects.
- `advanced-color-synthesizer`: Improve palette quality, color hierarchy, and contrast decisions.
- `interaction-motion-designer`: Improve motion language, transitions, and interaction feel.
- `layout-design-engine`: Improve layout systems, spacing rhythm, and composition structure.
- `adaptive-structure-architect`: Guide responsive structure and information hierarchy.
- `pattern-recognition-engine`: Analyze patterns and consistency across UI systems.
- `aesthetic-code-rules`: Apply aesthetic and maintainable UI code rules.

## Repository Structure

Each skill lives in its own folder and typically includes:

- `SKILL.md`: The main skill instructions and usage guidance.
- `agents/openai.yaml`: Metadata used by compatible tooling.

## Usage

Copy or install the needed skill folder into your Codex skills directory, then invoke it by name in your prompt when relevant.
